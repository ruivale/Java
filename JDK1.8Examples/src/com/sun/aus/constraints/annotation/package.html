Provides the core constraint annotations and classes to provide access to constraints.

The following diagram shows an overview of this package.

The core constraint annotations (e.g. {@link com.sun.aus.constraints.annotation.MaxLength MaxLength}) are those shown at the bottom with a stereotype of <<Constraint>>. The key class in this package (other than the constraints themselves) is {@link com.sun.aus.constraints.annotation.ConstrainedClass ConstrainedClass} which provides access to the property based constraints associated with a given class.

Note many of the constraints are similar to those in XML Schema of the same name.
